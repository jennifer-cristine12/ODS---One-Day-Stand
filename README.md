# ODS---One-Day-Stand
O projeto ODS é um projeto pessoal de desenvolvimento Front-end (HTML, CSS e Javascript)

##Problemas a serem solucionados

esse projeto serve como uma mini rede social para pessoas que vão passar o dia dos namorados sozinhos, podendo realizar coisas como:

- analsar as postagens sobre o que as pessoas fazem nesse tempo
- conversar com outras pessoas sejam individuais ou em grupo
- o algoritimo envia uma pessoa para conversar por um dia (por isso o nome)
